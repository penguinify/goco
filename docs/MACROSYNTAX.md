# TokenFunction
Possible Values:
- type <string> <ms as number>
- mouseset <number> <number>
- mouseclick <left|right|middle>
- mousemove <number> <number>
- keydown <string>
- keyup <string>
- delay <ms as number>
# Number
any positive integer, ex: delay 1000
# String
any string, ex: keydown "a".    
Must be double quoted.
# TokenKeyword
Possible Values:
- loop <number>
- forever
- end
